# SpiralCaster Playground Examples

This directory contains 23 Playground-generated example bundles from YouTube creator
SpiralCaster, collected from the video:
[The MOST DANGEROUS Pedal I've Ever Used: Polyend ENDLESS](https://youtu.be/nCYzbPEIBZo?si=Ehx_ZuWXTwMUoEj0)

Each effect folder typically includes:

- a compiled `.endl` file for Endless
- a PDF snapshot or descriptive card for the effect

These files are useful as a reference archive for:

- effect naming and categorization
- the range of results creators are getting from Playground
- future listening comparisons against hand-written SDK patches

There is no corresponding editable C++ source in this directory.
