# Playground Examples

Compiled Playground patches from other creators. These are useful for:

- listening reference
- studying patch naming and presentation
- building intuition about the kinds of effects Playground can generate

They are not source examples in the same way as `effects/*.cpp`.

This directory is separate from the official Polyend Plates archive. For Polyend's own
catalog of official Endless plates, attribution notes, and the local-only sync workflow,
see [`playground/polyend_plates/README.md`](../polyend_plates/README.md).
