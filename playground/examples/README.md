# playground/examples/

Ready-to-use `.endl` patches generated by Playground, organized by creator.

---

## Collections

| Directory | Creator | Effects | Notes |
|---|---|---|---|
| [`SpiralCaster_Examples/`](SpiralCaster_Examples/) | SpiralCaster | 23 | From the YouTube video [The MOST DANGEROUS Pedal I've Ever Used](https://youtu.be/nCYzbPEIBZo?si=Ehx_ZuWXTwMUoEj0); each effect includes `.endl` + prompt log PDF |

---

## Structure

Each effect lives in its own subdirectory:

```
SpiralCaster_Examples/
  EFFECT NAME/
    Effect Name.endl   ← drag onto Endless drive to load
    EFFECT NAME.pdf    ← prompt log: original prompt, any revisions, final result description
  ...
  README.md            ← collection overview
  CATALOG.md           ← quick-reference: all effects, knob assignments, footswitch behavior
```

---

## How to use an effect

1. Connect Endless via USB-C
2. Copy the `.endl` file onto the Endless drive
3. The effect loads automatically

Refer to the matching `.pdf` for control details, or see
[`SpiralCaster_Examples/CATALOG.md`](SpiralCaster_Examples/CATALOG.md) for a compact
reference table covering all 23 effects.

---

## Adding your own

To add effects from another creator, create a new subdirectory here and add a short
README explaining the source and any relevant context. If you have prompt logs, include
them — they're the most useful reference material for understanding why an effect sounds
the way it does.
